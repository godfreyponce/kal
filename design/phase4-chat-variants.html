Three directions for Chat

Same conversation, three layouts. The chat is ephemeral (fresh session each open, no thread list) and streams replies. Its signature element is the tool card with Undo — when the assistant logs food or marks a meal eaten, it shows what changed and lets you revert the whole write batch in one tap. Pick a direction (or mix) and I'll build it in React.

Variant A

Transcript

Flat, document-style. No bubbles — messages read like a log; tool actions are ledger dividers with inline Undo. Calmest, closest to the Today screen.
