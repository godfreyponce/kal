Plan — 3 editor directions

The first editing UI for the three things every screen reads but nothing can change: the profile, the meal-plan template, and Kal's memory facts. All data here is the public demo/seed plan. Two shared rules in every variant: edits change the everyday template (day-scoped chat adjustments ⇄ are untouched), and daily targets are derived from the plan — each variant shows its own treatment of that recalculation.

Tell me which direction (or which pieces of each) and I'll refine it into plan-combined.html.

Variant A
Ledger
One scroll, everything visible, edit-in-place: tap any number and it becomes an input right there. No forms, no navigation. Targets sit between profile and plan as a “derived” strip that recalcs live.
