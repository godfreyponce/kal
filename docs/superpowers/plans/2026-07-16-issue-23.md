**Goal:** Swiping up on the weekly-adherence strip on `/plan` (or tapping a small "history" row) pulls up a bottom sheet with a month calendar of adherence history, pageable back to the first logged month.

**Architecture:** All history is fetched server-side in `/plan`'s existing `Promise.all` (single group-by-date aggregate over `log_entries`, judged in memory) and passed down as props; the client pages months locally with pure date math — no new API route. The sheet reuses the #24 `.sheet-*` shell CSS and `lib/sheet-gesture.ts` math; the swipe-up gesture arms on the strip's pointerdown but tracks moves on `window`.

**Tech Stack:** Next 16 App Router (server component + one client component), Drizzle/neon-http, vitest.

## Global Constraints

- Owner-approved design: **Heat variant** in `design/plan-adherence-calendar-variants.html` (owner-redefined 2026-07-16). Match it visually: 32×30px chips, radius 9px, mono 10px day numbers, serif month title, grabber, legend.
- **Semantics (owner rule, exact):** dark green `--green-tx` = nutrients hit eating only meal-plan food; light green `--green-bg` = nutrients hit but day included non-plan food ("extras"); red `rgba(159,47,45,.45)` = miss, single tone, **and a not-logged past day IS a miss**. Today = accent outline, never judged. Future = faint numbers. Days before the first logged day = bare numbers ("pre"). Both greens count as "on plan" in the summary; streaks break on red (miss or unlogged), not on today.
- "A day" is a civil date string from `todayInAppTz()` (`lib/time.ts`, APP_TZ America/Chicago); all month math is UTC arithmetic on such strings (convention in `lib/adherence.ts:24-34`). Never bare `new Date()` day math.
- UI copy: no em dashes, no middle-dot separators (owner rule). Summary copy exactly: `` `${on} of ${judged} judged days on plan, best streak ${best}` ``; empty month: `no judged days yet`.

```ts
export type CalDayState = JudgedState | "unlogged" | "today" | "future" | "pre";
export interface JudgedDay { date: string; state: JudgedState }
export interface AdherenceHistory { days: JudgedDay[]; firstLogDate: string | null }
export interface HistoryRow { date: string; kcal: number; proteinG: number; hasExtras: boolean }
export interface CalCell { day: number; date: string; state: CalDayState }

  leading: number;             // blank cells before day 1 (Mon-first)

export function classifyHistoryRows(rows: HistoryRow[], targets: Macros): JudgedDay[];
export function historyMonthRange(history: AdherenceHistory, today: string): { first: number; last: number }; // index = year*12 + month0
```

```ts
// Pure calendar model for the #23 adherence history sheet. DB-free on purpose:
// tests need no DATABASE_URL (same split as adherence-view.ts vs adherence.ts).
import { judgeDay, type Macros } from "./adherence-view";

/* ...types from the Interfaces block above, verbatim... */

const MONTHS = ["January","February","March","April","May","June","July","August","September","October","November","December"];

export function classifyHistoryRows(rows: HistoryRow[], targets: Macros): JudgedDay[] {
  return rows.map(r => ({
    date: r.date,
    state: judgeDay({ ...zeroMacros, kcal: r.kcal, proteinG: r.proteinG }, targets)
      ? (r.hasExtras ? "onx" : "on")
      : "off",
  }));
}
// zeroMacros: fill whatever other fields Macros requires with 0 — check the real type.

export function historyMonthRange(history: AdherenceHistory, today: string): { first: number; last: number } {
  const idx = (s: string) => { const d = new Date(s + "T00:00:00Z"); return d.getUTCFullYear() * 12 + d.getUTCMonth(); };
  const last = idx(today);
  return { first: history.firstLogDate ? Math.min(idx(history.firstLogDate), last) : last, last };
}

export function buildCalMonth(monthIdx: number, history: AdherenceHistory, today: string): CalMonth {
  const y = Math.floor(monthIdx / 12), m = monthIdx % 12;
  const daysInMonth = new Date(Date.UTC(y, m + 1, 0)).getUTCDate();
  const leading = (new Date(Date.UTC(y, m, 1)).getUTCDay() + 6) % 7;   // Mon-first
  const byDate = new Map(history.days.map(d => [d.date, d.state]));
  const cells: CalCell[] = [];
  for (let day = 1; day <= daysInMonth; day++) {
    const date = `${y}-${String(m + 1).padStart(2, "0")}-${String(day).padStart(2, "0")}`;
    let state: CalDayState;
    if (date === today) state = "today";
    else if (date > today) state = "future";
    else if (!history.firstLogDate || date < history.firstLogDate) state = "pre";
    else state = byDate.get(date) ?? "unlogged";
    cells.push({ day, date, state });
  }
  let on = 0, judged = 0, streak = 0, bestStreak = 0;
  for (const c of cells) {
    if (c.state === "on" || c.state === "onx") { on++; judged++; streak++; bestStreak = Math.max(bestStreak, streak); }
    else if (c.state === "off" || c.state === "unlogged") { judged++; streak = 0; }
    // today/future/pre: neither judged nor streak-breaking
  }
  return { label: `${MONTHS[m]} ${y}`, leading, cells, summary: { on, judged, bestStreak } };
}
```

```ts
useEffect(() => {
  const strip = stripRef.current;
  if (!strip || window.matchMedia("(prefers-reduced-motion: reduce)").matches) return;
  let y0 = 0, t0 = 0, armed = false, active = false;
  const down = (e: PointerEvent) => { y0 = e.clientY; t0 = e.timeStamp; armed = true; active = false; };
  const move = (e: PointerEvent) => {
    if (!armed) return;
    const dy = e.clientY - y0;
    if (!active) {
      if (dy < -6) { active = true; openPulled(); }        // mount sheet in dragging state
      else if (dy > 12) { armed = false; return; }          // clearly a downward scroll
      else return;
    }
    const card = cardRef.current; if (!card) return;
    const h = card.offsetHeight, pull = Math.min(-dy, h);
    card.style.setProperty("--sheet-y", `${h - pull}px`);
    card.style.setProperty("--scrim-o", String(pull / h));  // match how the shell CSS reads it
  };
  const upOrCancel = (e: PointerEvent) => {
    if (!armed) return;
    armed = false;
    if (!active) return;
    active = false;
    const dy = y0 - e.clientY, h = cardRef.current?.offsetHeight ?? 1;
    const v = dy / Math.max(1, e.timeStamp - t0);
    settlePull(dy > h * 0.3 || v > 0.5);                    // true -> open, false -> sink back
  };
  // iOS: pointermove preventDefault does NOT stop scrolling; this non-passive
  // touchmove is what actually blocks the page from scrolling once we claim.
  const touch = (e: TouchEvent) => { if (active) e.preventDefault(); };
  strip.addEventListener("pointerdown", down);
  window.addEventListener("pointermove", move);
  window.addEventListener("pointerup", upOrCancel);
  window.addEventListener("pointercancel", upOrCancel);
  strip.addEventListener("touchmove", touch, { passive: false });
  return () => { /* remove all five */ };
}, [stripRef]);
```

`openPulled()` mounts the sheet with `dragging` (no enter transition); `settlePull(openIt)` clears `dragging` and either adds `open` or plays the exit and unmounts. Add `user-select:none` (and `-webkit-`) to the strip while armed or via `.adh-body{user-select:none}`. Day-cell taps stay safe: claiming requires >6px movement, and no pointer capture happens before the claim.
