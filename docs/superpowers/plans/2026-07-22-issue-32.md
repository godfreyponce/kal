**Goal:** Apply the #28 round-1 pick (V2 + two owner removals) to the Today screen, per the spec
`design/today-silent-menu-combined.html`, and promote the silent-menu palette from
Groceries-scoped `--gro-*` tokens to shared `:root` `--sm-*` tokens.

**Architecture:** Pure restyle — no data, route, or lib changes. Two tasks on a ticket branch:
(1) token promotion with Groceries switched to consume the shared set, (2) Today markup + CSS
rewrite. All class restyles are Today-scoped (verified: no other screen uses `.mrow`, `.seg`,
`.ct`, `.ck`, `.box`, `.checklist`, `.mchev`, `--ring-track`, `--seg-empty`). Shared things —
`.chat-link` (Plan back link), `.kicker` (weigh-in, Chat), `.rule` (Plan), `.mpop-*`/`.sheet-*`
(Plan modals), `--protein/--carbs/--fat` (popup, Plan, Chat) — are left untouched; Today gets
new class names or scoped values instead.

**Tech Stack:** Next.js App Router (Next 16), plain CSS in `app/globals.css`, no Tailwind
utility classes in these files.

  hard-refresh before judging visuals.

  loads via `db/env`.

## Locked design decisions (read before building)

- **Meals band background is `#f1e7d3` (mockup literal), NOT `var(--sm-band)` (#f6efe2).**
  The shared band token #f6efe2 is nearly invisible on the cream canvas #f7f6f3 — that is why
  the picked V2 mockup deliberately uses the darker #f1e7d3 for the band while using #f6efe2
  for done-checkbox fills. Groceries' band keeps #f6efe2 (via the token) because its canvas
  is white. Do not "unify" these.
- **Macro colors are passed as scoped `var(--sm-*)` values from `page.tsx`**, not by redefining
  `--protein/--carbs/--fat` — those tokens color the meal popup, Plan's modal bars, and Chat,
  which this ticket must not restyle.
- **`--ring-track` and `--seg-empty` are redefined in `:root` to `#eee5d2`** (warmed cream).
  Both are consumed only by Today (verified by grep), so this spills nowhere.
- **Today's nav gets a new `.top-link` class**; `.chat-link` CSS stays for Plan's back link.
- **The kcal chevron (`.mchev`) is removed and the pill reads "815 cal"** — the mockup shows
  pills with a "cal" suffix and no chevron; the row stays tappable (whole `.rowbtn` opens the
  popup). "Chat →" becomes "Chat ›" per the mockup.
- **`.adjmark` (⇄) inside the pill is recolored to cream** (`var(--sm-band)`) — terracotta on
  the chocolate pill is illegible. The popup's terracotta adjmark is untouched.
- **Reduced motion turns the enter stagger fully off** (`animation: none`), matching the
  mockup and the Done-when's "off-switch" wording (unlike Groceries' fade-only variant).
